### 安装 anaconda

```shell
wget https://repo.anaconda.com/archive/Anaconda3-2023.09-0-Linux-x86_64.sh
sh Anaconda3-2023.09-0-Linux-x86_64.sh
echo 'export PATH=/root/anaconda3/bin:$PATH'>> ~/.bashrc
source ~/.bashrc
conda env list
rm Anaconda3-2023.09-0-Linux-x86_64.sh
```
