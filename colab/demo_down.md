### git lfs 安装
```text
wget https://github.com/git-lfs/git-lfs/releases/download/v3.2.0/git-lfs-linux-amd64-v3.2.0.tar.gz
tar -xzf git-lfs-linux-amd64-v3.2.0.tar.gz
PATH=$PATH:/home/liuchanghong/git-lfs-3.2.0
git lfs install
git lfs version
```
